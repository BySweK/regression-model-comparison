Manufacturing Quality PredictionThis project focuses on predicting product quality ratings based on manufacturing parameters using various regression algorithms. The primary goal was to explore the impact of different machine learning models and data preprocessing techniques on predictive accuracy.  Project OverviewThe dataset contains manufacturing data, including temperature, pressure, and various material metrics. In this project, I performed the following steps:  Exploratory Data Analysis (EDA): Investigated correlations and removed highly redundant features (e.g., 'Temperature x Pressure') to prevent overfitting.  Data Preprocessing: Standardized features using StandardScaler to ensure optimal performance for distance-based algorithms.  Pipeline Development: Implemented Pipeline to integrate feature scaling and PolynomialFeatures to handle non-linear relationships efficiently.  Model Comparison: Evaluated and compared the performance of several regression models:  Linear RegressionPolynomial RegressionSupport Vector Regressor (SVR)K-Nearest Neighbors (KNN)Decision Tree RegressorResultsThe project demonstrates that while simple linear models may fail to capture complex patterns in manufacturing data, applying polynomial features significantly boosts predictive power.
